# Customer Retention Dashboard

## Project Overview
This project analyzes customer churn behavior in a European bank dataset using Excel, Python, and Streamlit.

## Objectives
- Analyze customer churn patterns
- Identify factors influencing retention
- Visualize customer demographics and behavior
- Build an interactive dashboard

## Tools Used
- Microsoft Excel
- Python
- Pandas
- Plotly
- Streamlit
- GitHub

## Key Findings
- Female customers showed higher churn rates.
- Germany had the highest churn rate among all regions.
- Active customers were less likely to leave the bank.
- Customers with 3-4 products showed higher churn rates.
- Churned customers had higher average balances.
